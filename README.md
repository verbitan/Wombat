Wombat Colorscheme for Vim
=============================

My fork of the wombat colour scheme by Lars H. Nielsen <dengmao@gmail.com>

Installation
------------

Move `wombat.vim` to your `.vim/colors` directory. After downloading the repository:

        $ cd Wombat/colors
        $ mv wombat.vim ~/.vim/colors/
